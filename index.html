Live Data Dashboard

Live Data Streamer is a real-time data visualization dashboard that streams and displays high-frequency data (100 samples/sec, 10 values per sample) from a TCP server. Built with Python (backend) and React + Recharts (frontend), it leverages WebSockets for efficient real-time data transmission and dynamic visualization.




MQTT Chat App

This project uses MQTT to enable real-time messaging and file sharing between clients. It supports direct messaging, client lookup, and offline message storage, ensuring reliable communication and seamless file transfers.
